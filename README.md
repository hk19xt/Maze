# Maze
This program is finding the fastest path using 2D ASCII data map.

How to run this program:
1) The maps will be represented by S,E,O,X: S (Start), E (Exit), O (Floor), X (Wall).
2) The map (.txt files) will be accepted by System.in.
3) The maze is passed by up, down, east, north, south, west. 
4) The input files consist of a width, height, and depth, followed by a map.

This program is one of individual school assignments using Java.
